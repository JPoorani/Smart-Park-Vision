# Smart Park Vision 

Smart Park Vision is an intelligent parking management system using computer vision.

## Technologies Used
- Python
- YOLO Object Detection
- OpenCV
- HTML, CSS

## Working
- Camera monitors parking area
- Parking slots are marked using bounding boxes
- YOLO detects vehicles
- If vehicle is inside box → occupied
- Else → Free Space
- Count updates automatically

## Features
- Real-time vehicle detection
- Smart parking system
- Automatic slot counting
